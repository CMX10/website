Install

  • Download Latest ROM and put zip file in SDCARD
  • Download Kernel, and Flash using Flastool.
  • DO NOT USE kernel.sin file provided by ROM
  • Start phone, when purple LED appears, press the back button. You are now in RecoveryMode.
  • Wipe Cache/Data
  • Select Install zip and browse to sdard card and to the ROM.zip file. Install
  • Once installation is done, reboot phone.
  • Let phone boot completely
  • Download gapps and put them on SDCARD
  • Reboot phone into recovery
  • Select install from zip and browse to gapps
  • Reboot phone. You are now Done.

Build From sources

  • repo init http://github.com/CyanogenMod/android.git -b jellybean
  • repo sync -j 18
  • cd devices
  • mkdir semc
  • cd semc
  • git clone http://github.com/CMX10/android_device_semc_es209ra -b jellybean es209ra
  • cd ../../vendor
  • mkdir -p semc
  • cd semc
  • git clone http://github.com/CMX10/proprietary_es209ra -b jellybean es209ra
  • cd ../cm/
  • .get-prebuilts
  • cd ..
  • . build/envsetup.sh
  • brunch es209ra
  • The current generated rom isn't flashable.
  • extract META-INF/com/google/android/update-binary from latest rom and replace into your ROM.
